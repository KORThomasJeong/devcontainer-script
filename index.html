02

빠른 시작

STEP 01

Docker 실행

Docker Desktop을 실행하고 엔진이 정상 동작하는지 확인합니다.

docker info
STEP 02

스크립트 실행

레포 루트에서 start.sh를 실행합니다. 샘플 선택 → 포트 자동 확인 → 컨테이너 시작까지 자동 처리됩니다.

./start.sh
STEP 03

VS Code로 연결

VS Code에서 Cmd+Shift+P → Dev Containers: Attach to Running Container 선택 후 개발 시작.

03

Claude Code 사용

컨테이너 안에서 바로 Claude Code 사용

모든 샘플에는 postCreateCommand로 Claude Code CLI가 자동 설치됩니다.
컨테이너 터미널에서 아래 명령어로 즉시 AI 코딩을 시작할 수 있습니다.
최초 실행 시 Anthropic 계정 인증 또는 API 키를 입력합니다.

claude
